# SOC Incident Response Labs

A collection of hands-on Security Operations Center (SOC) and Digital Forensics & Incident Response (DFIR) labs completed in a virtual environment.

These labs demonstrate practical incident response workflows using industry tools and threat intelligence platforms.

## Objectives

- Practice real-world incident response procedures
- Investigate Indicators of Compromise (IoCs)
- Document security incidents
- Perform post-incident reviews
- Build a professional cybersecurity portfolio

## Skills Demonstrated

- Incident Response
- Threat Hunting
- Threat Intelligence
- IOC Analysis
- VirusTotal
- AbuseIPDB
- Docker
- Linux Administration
- Documentation
- Security Operations (SOC)
